Formule de Kyungu

La formule de Kyungu est une expression mathématique originale pour la transformée de Laplace inverse, proposée par Pathy Kyungu, professeur de mathématiques congolais. Elle exprime la fonction inverse $f(t)$ à partir des dérivées successives d’une fonction auxiliaire $\varphi(x) = F(1/x)$.

Définition

Soit $F(p)$ une fonction dans le domaine de Laplace. On définit :

\[ \varphi(x) = F\left(\frac{1}{x}\right) \] \[ f(t) = \varphi(0)\delta(t) + \sum_{n=1}^{\infty} \frac{\varphi^{(n)}(0)}{(n!)^2} \cdot n \cdot t^{n-1} \]

Formule Kyungu–Mellin–Laplace (KML)

La formule est reliée à la transformée de Mellin inverse via la substitution $t = -\ln x$ :

\[ \mathcal{M}^{-1}[F](x) = \mathcal{L}^{-1}[F](-\ln x) \]

Exemples

  • $F(p) = \dfrac{1}{p^4}$ ⇒ $f(t) = \dfrac{1}{6}t^3$
  • $F(p) = \dfrac{1}{1 + p^2}$ ⇒ $f(t) = \sin(t)$
  • $F(p) = \dfrac{p^2}{1 + p^3}$ ⇒ \[ f(t) = \frac{1}{3}e^{-t} + \frac{2}{3} e^{t/2} \cos\left(\frac{\sqrt{3}}{2}t - \frac{\pi}{3}\right) \]
